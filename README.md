# docker-plone
Plone on Docker

## Building container
```
$ docker build -t thobalose/docker-plone .
```

## Running container
```
$ docker run -d -p 8080:8080 --name plone thobalose/docker-plone
```

and point your browser to: 

  * [localhost:8080](http://localhost:8080)
